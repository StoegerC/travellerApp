# Traveller Charakterdatenblatt

Inoffizielles digitales Charakterblatt für das Pen-&-Paper-Rollenspiel **Traveller Classic**.
Entwickelt für den Einsatz am Spieltisch — primär auf iPad und iPhone, auch auf Desktop nutzbar.

## Features

- **Charakter** — Portrait (mehrere), Name, Titel, Heimatplanet, Alter
- **Attribute & Skills** — 6 Grundattribute mit Wunden-Tracking, vollständige Skillliste
- **Ausrüstung** — Waffen, Rüstung, Ausrüstungsgegenstände
- **Kampf** — Trefferpunkte, Initiative, Waffenwerte, Erste Hilfe
- **Finanzen** — Kontostand, Transaktionen, wiederkehrende Kosten, Schulden
- **Werdegang** — Karriere-Terme, Schlüsselereignisse, Hintergrundgeschichte
- **Log** — Sitzungsjournal, Personen, Orte, Quests mit gegenseitiger Verlinkung
- **Karte** — Travellermap-Integration mit Ortsverknüpfungen
- **Dark Mode**, Autosave, Versionsverlauf, JSON-Export/Import

## Technologie

Reines Vanilla JavaScript — kein Framework, kein Build-Tool, keine externen Abhängigkeiten.  
Läuft vollständig offline im Browser. Daten werden lokal in IndexedDB gespeichert.

## Nutzung

Direkt öffnen: `frontend/index.html` im Browser — oder als PWA installieren:

- **iPhone / iPad**: URL in Safari öffnen → Teilen → „Zum Home-Bildschirm"
- **Mac**: URL in Chrome/Edge öffnen → Installations-Symbol in der Adressleiste
